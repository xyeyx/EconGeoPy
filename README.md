# EconGeoPy
A collection of utilities for computing indices in economic geography.

Working in progress... Aim to provide a python package *EcGeoPy* that provides comparable functionality as the well-known [Economic Geography Toolkit](https://github.com/PABalland/EconGeo "PABalland/EconGeo") by [Pierre-Alex Balland](https://paballand.com), but of course in Python instead of R...
